# SevenEngine
* 一个小玩具
## 构建
* 安装cmake（3.6或更高版本）
* 安装visualstudio（2022或更高版本）
* 运行OpenProject.bat（如果已经成功生成项目会打开相应的sln，否则会生成相应的项目后打开sln）
* 当需要重新生成项目时，可以运行GerneateProjectFile.bat，如果需要强制刷新，可以增加-force命令
